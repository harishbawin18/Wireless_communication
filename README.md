# Digital Communication System Simulations using MATLAB Simulink 📡

## Overview

This repository contains MATLAB Simulink models developed to simulate various **digital communication system techniques** and wireless channel effects.

Each Simulink file represents an individual communication system or signal processing model used to study modulation, fading channels, diversity techniques, and equalization methods.

These simulations help in understanding practical communication system behavior under different channel conditions.These simulations also have eye diagram and constellation diagram for performance analysis.
To get the proper throughput the BER calulation is also include for all the simulations.

---

## Repository Contents

Each file in this repository represents a separate Simulink model implementing a specific communication technique.

### Implemented Simulink Models

* **BPSK (2).slx**
  Simulation of Binary Phase Shift Keying (BPSK) modulation and demodulation.

* **Equaliser 1.slx**
  Channel equalization model used to compensate for signal distortion.

* **RAY_RIC.slx**
  Simulation of Rayleigh and Rician fading channel models.

* **diversity.slx**
  Implementation of diversity techniques to improve signal reliability.

* **multipathfadig.slx**
  Multipath fading channel simulation to study signal propagation effects.

* **ofdm.slx**
  Orthogonal Frequency Division Multiplexing (OFDM) communication system simulation.

* **rb.slx**
 BPSK modulated signal studied using rayliegh fading in awgn channel.

* **stbc (2).slx**
  Space-Time Block Coding (STBC) implementation for wireless communication.

* **wcb2.slx**
  QPSK modulation technique analysed using rayliegh fading channel and awgn channel.
